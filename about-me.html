about me

I work with technologies such as HTML5, CSS3, JavaScript and others in order to create customized, functional and visually appealing web solutions, that not only help realize the marketing and branding strategies of my clients, but that also form the foundation for their internet presence.

I'm never satisfied with how much I know. I take every challenging encouter as an opportunity to learn and increase my skill set, thereby increasing my efficiency and adding to my ability to provide more options to the web solutions of my clients.

workflow

My web development workflow follows three stages... pre-production, production and post production. During pre-production, I listen to the needs of the client and use their input to design a visual mock-up for their site.

With design parameters defined, production begins. Here, I set up my development environment, code the site's skeleton and stylings using HTML5 and CSS3, and then I add functionality and interaction using JavaScript and JQuery. I use Github for version control and back-up.

In post-production I take care of final tweaks and set up the hosting environment that I will use to host and maintain the client's site.
